
# 🌿 Personal Portfolio — *Nature-Inspired & Modern*

Welcome to my personal portfolio site — a clean and modern showcase of who I am, the tools I use, and the projects I’ve built. This project is designed with both aesthetics and functionality in mind, offering a smooth browsing experience and a secure admin interface for easy content management.

---

## 🌐 Frontend Routes

### `/` — Home

* A welcoming landing page featuring:

  * A personal photo and **About Me** section
  * A **Skills** section
  * A **Contact** form
  * A dynamic list of **Project Cards** displayed in no particular order

### `/admin` — Admin Dashboard

* 🔒 **Secured route** (requires both authentication and a password)
* Features:

  * Create, edit, or delete portfolio projects
  * Update personal data like profile image, skills, contact info, and bio
  * Manage visibility of content directly from the dashboard

### `/project/:id` — Project Details

* Clicking on a project card leads to a detailed page for that project
* Includes extended information such as description, tools used, live demo, and GitHub links
